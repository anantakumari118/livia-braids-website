# **Livia’s Braids – Salon Website**

A modern, responsive braids studio website built using **React + TypeScript + Tailwind CSS**.
Includes service pricing, gallery, booking options, and a clean dark UI designed for a real braiding salon.

## **Features**

* Responsive layout (mobile-first)
* Modern dark theme with smooth animations
* Services with detailed pricing
* Gallery with style numbers
* WhatsApp quick-booking button
* Scroll-to-top navigation
* React Router pages (Home, Services, Gallery, About, Visit, Contact)

## **Tech Stack**

* React (Vite)
* TypeScript
* Tailwind CSS
* React Router DOM

## **Setup**

```bash
npm install
npm run dev
```

## **Build**

```bash
npm run build
```
